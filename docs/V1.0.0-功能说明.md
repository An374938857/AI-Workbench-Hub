# V1.0.0 功能说明

## 版本声明
- V1.0.0

## 文档使用说明

- 章节结构固定为：`模块定位与适用角色`、`典型使用场景`、`功能清单`、`操作路径`、`输入/输出与结果表现`、`权限与可见性`、`约束与边界`、`常见问题（FAQ）`。
- 能力描述统一句式：`能做什么 + 在哪做 + 谁可做 + 做完看到什么`。
- 角色定义：
  - `普通用户`：日常业务使用者。
  - `技能管理员（Provider）`：可进入管理后台，重点管理 Skill。
  - `超级管理员（Admin）`：拥有全部管理能力。

## 术语表
- `Skill`：平台内可被调用的能力单元，包含说明、技能包、版本与状态。
- `MCP`：外部工具能力接入单元，可由 AI 在对话中调用。
- `提示词模板`：可复用的对话上下文模板，支持默认、收藏、版本回退。
- `流程定义`：项目/需求可复用的节点化流程模板。
- `流程实例`：绑定到具体项目或需求后的运行态流程。
- `资料（Asset）`：挂载在流程节点上的文件、URL 或 Markdown 内容。
- `会话沙箱`：会话执行中生成或上传的文件工作区。
- `引用治理`：会话中对资料引用范围和选择状态的管理能力。

## 1. 首页
![首页模块截图](/Users/bk27284/Documents/ai-skill-sharing-platform/docs/images/v1.0.0/01-home.png)

### 模块定位与适用角色
- 首页是统一对话入口，面向全部登录角色。

### 典型使用场景
- 新建一次带模型、模板、技能预设的对话。
- 快速上传附件并发起 AI 协作。
- 使用斜杠命令（如模型/模板/技能/MCP）完成快速切换。

### 功能清单
- 能创建新对话，在首页输入区，普通用户/管理员可用，创建后跳转对话页。
- 能选择模型，在首页模型选择器，普通用户/管理员可用，提交后会话使用选中模型。
- 能选择提示词模板，在首页模板选择器，普通用户/管理员可用，提交后会话带入模板。
- 能选择技能，在首页技能列表，普通用户/管理员可用，会话初始能力随之绑定。
- 能查看 MCP 列表，在首页命令面板，普通用户/管理员可用，便于确认可调用工具能力。
- 能上传附件（含图片预览、拖拽、粘贴），在首页输入区，普通用户/管理员可用，附件随首条消息进入会话。
- 能优化提示词，在输入区优化按钮，普通用户/管理员可用，弹窗展示优化结果后可回填。
- 能调出快捷键提示菜单，在任意主界面按 `Command+K`（Windows/Linux 对应 `Ctrl+K`），登录用户可用，显示全局快捷操作说明。
- 能调出对话全局搜索，在任意主界面按 `Command+F`（Windows/Linux 对应 `Ctrl+F`），登录用户可用，快速定位历史会话。
- 能唤起快捷指令面板，在输入框输入 `/`，登录用户可用，显示模型/模板/技能/MCP 等指令补全。

### 操作路径
- 顶部导航进入首页 `/`。
- 在输入框填写内容，按需选择模型/模板/技能后发送。
- 通过 `/` 触发命令面板快速切换能力。

### 输入/输出与结果表现
- 输入：自然语言、附件、命令、模型/模板/技能选择。
- 输出：新会话、首轮消息、附件绑定、可继续追问的会话上下文。

### 权限与可见性
- 登录用户可访问。
- 能看到的数据范围受用户权限与后端配置约束。

### 约束与边界
- 上传附件受格式与大小限制。
- 未配置可用模型或模板时，部分能力入口可见但不可用。

### 常见问题（FAQ）
- Q：未选模板会怎样？
- A：系统使用默认模板（若配置了全局默认）。
- Q：为什么某些模型不可选？
- A：模型可能被管理员停用，或角色无访问权限。

## 2. 项目
![项目模块截图](/Users/bk27284/Documents/ai-skill-sharing-platform/docs/images/v1.0.0/02-projects.png)

### 模块定位与适用角色
- 项目模块用于管理项目台账、绑定流程、推进节点并沉淀资料，面向业务执行角色。

### 典型使用场景
- 创建项目并指定负责人与优先级。
- 为项目绑定流程定义并驱动节点推进。
- 在节点上沉淀资料和关联会话。

### 功能清单
- 能创建/编辑/删除项目，在项目列表页，普通用户/管理员可用，列表即时刷新。
- 能设置项目等级、负责人、周期，在项目表单，普通用户/管理员可用，项目元数据可追踪。
- 能绑定流程定义，在项目详情操作区，普通用户/管理员可用，生成或关联流程实例。
- 能推进节点状态，在流程节点操作区，普通用户/管理员可用，节点状态更新并影响整体流程状态。
- 能绑定/解绑会话，在节点会话面板，普通用户/管理员可用，形成“节点-会话”协作链路。
- 能在节点上传资料（文件/URL/Markdown），在节点资料面板，普通用户/管理员可用，资料进入可检索资产池。
- 能预览、下载、删除、重抓资料，在节点资料历史区，普通用户/管理员可用，资料状态可追踪。
- 能查看项目关联沙箱视图，在项目沙箱抽屉，普通用户/管理员可用，集中查看相关文件产出。
- 能在项目流程节点使用文件沙箱，在项目节点上下文中，普通用户/管理员可用，支持文件沉淀与后续协作复用。

### 操作路径
- 顶部导航进入 `/projects`。
- 通过“新建项目”填写基础信息。
- 在项目行操作中进行流程绑定、节点推进、资料与会话管理。

### 输入/输出与结果表现
- 输入：项目基本信息、流程选择、节点操作、资料内容。
- 输出：项目台账、流程运行状态、节点资料沉淀、会话关联关系。

### 权限与可见性
- 普通用户与管理员可编辑。
- 受限角色可浏览但不可执行敏感操作（依后端鉴权为准）。

### 约束与边界
- 节点状态流转受流程定义约束。
- URL 资料抓取受外部站点可访问性影响。

### 常见问题（FAQ）
- Q：项目能否先不绑流程？
- A：可以，状态显示为未绑定流程。
- Q：节点资料上传后为何还在“处理中”？
- A：外部抓取/解析任务异步执行，完成后会自动更新状态。

## 3. 需求池
![需求池模块截图](/Users/bk27284/Documents/ai-skill-sharing-platform/docs/images/v1.0.0/03-requirements.png)

### 模块定位与适用角色
- 需求池用于需求条目管理与交付流程协同，面向产品与执行团队。

### 典型使用场景
- 创建需求并指定优先级、负责人、截止日期。
- 为需求绑定流程并按节点推动落地。
- 在需求节点沉淀资料与对话证据。

### 功能清单
- 能创建/编辑/删除需求，在需求池页面，普通用户/管理员可用，需求条目可检索。
- 能按项目、优先级、状态、负责人筛选，在筛选栏，普通用户/管理员可用，快速定位目标需求。
- 能绑定流程定义，在需求详情操作区，普通用户/管理员可用，需求进入可执行流程。
- 能推进节点，在节点操作区，普通用户/管理员可用，流程进度同步变化。
- 能绑定/解绑会话，在节点会话面板，普通用户/管理员可用，形成需求决策记录。
- 能上传与维护节点资料，在资料面板，普通用户/管理员可用，资料可预览、下载、删除、重抓。
- 能查看需求关联沙箱，在需求沙箱抽屉，普通用户/管理员可用，统一查看文件产物。
- 能在需求流程节点使用文件沙箱，在需求节点上下文中，普通用户/管理员可用，支持资料沉淀与检索协作。

### 操作路径
- 顶部导航进入 `/requirements`。
- 点击“创建需求”，填写项目、标题、优先级等信息。
- 在需求行内执行流程与资料操作。

### 输入/输出与结果表现
- 输入：需求标题、描述、负责人、优先级、节点操作、资料内容。
- 输出：需求全生命周期状态、流程与资料关联记录。

### 权限与可见性
- 普通用户与管理员可编辑。
- 可见数据范围受账号权限和后端策略控制。

### 约束与边界
- 未绑定流程时无法执行节点推进。
- 资料处理速度受文件规模与外部源稳定性影响。

### 常见问题（FAQ）
- Q：需求可否跨项目迁移？
- A：当前通过编辑需求的项目归属实现调整（以权限允许为前提）。
- Q：为何无法推进某个节点？
- A：可能是前置节点未完成或流程状态不允许。

## 4. 能力广场（Skills）
![能力广场模块截图](/Users/bk27284/Documents/ai-skill-sharing-platform/docs/images/v1.0.0/04-skills-square.png)

### 模块定位与适用角色
- 能力广场用于浏览和选择已发布 Skill，面向全部登录用户。

### 典型使用场景
- 按标签或关键词找到适合当前任务的 Skill。
- 查看 Skill 详情、功能说明、示例和更新日志后再使用。

### 功能清单
- 能分页浏览已发布 Skill，在 `/skills`，全部登录用户可用，卡片展示核心信息。
- 能按关键词检索 Skill，在搜索框，全部登录用户可用，结果即时过滤。
- 能按场景标签筛选，在标签条，全部登录用户可用，聚焦特定能力类型。
- 能按个人排序偏好拖拽排序，在卡片区，全部登录用户可用，排序偏好自动保存。
- 能查看 Skill 详情（说明、SKILL.md、示例、更新日志），在 `/skills/:id`，全部登录用户可用，辅助选择。

- 顶部导航进入 `/skills`。
- 点击 Skill 卡片进入详情页。

- 输入：搜索词、标签筛选、排序拖拽。
- 输出：符合条件的 Skill 列表和详细能力说明。

### 权限与可见性
- 登录用户可见。
- 未发布/下架 Skill 不在广场展示。

### 约束与边界
- 列表仅展示当前账号可见的已上线能力。

### 常见问题（FAQ）
- Q：为什么看不到某个 Skill？
- A：可能未发布、已下架或当前角色无权限。

## 5. MCP 广场
![MCP广场模块截图](/Users/bk27284/Documents/ai-skill-sharing-platform/docs/images/v1.0.0/05-mcp-square.png)

### 模块定位与适用角色
- MCP 广场用于查看平台已接入的外部工具能力，面向全部登录用户。

### 典型使用场景
- 了解 AI 可调用哪些外部工具。
- 对目标 MCP 做连通性测试（有权限时）。

### 功能清单
- 能浏览 MCP 列表，在 `/mcps`，登录用户可用，显示描述、工具数、健康状态。
- 能按关键词筛选 MCP，在搜索框，登录用户可用，快速定位目标工具集。
- 能查看 MCP 详情与工具列表，在 `/mcps/:id`，登录用户可用，清楚可调用范围。
- 能执行 MCP 测试，在卡片测试动作，具备权限用户可用，返回连接结果与耗时。
- 能按个人偏好拖拽排序，在列表区，登录用户可用，排序结果可持久化。

### 操作路径
- 顶部导航进入 `/mcps`。
- 点击卡片进入详情，查看工具定义。

### 输入/输出与结果表现
- 输入：关键词、测试动作。
- 输出：MCP 可用性状态、工具数量与定义信息。

### 权限与可见性
- 登录用户可见公开 MCP。
- 部分测试能力受角色和配置控制。

### 约束与边界
- 连通性受外部服务与凭据配置影响。

### 常见问题（FAQ）
- Q：测试失败是否代表功能不可用？
- A：通常代表当前配置或网络不可达，需要管理员排查。

## 6. 提示词模板
![提示词模板模块截图](/Users/bk27284/Documents/ai-skill-sharing-platform/docs/images/v1.0.0/06-prompt-templates.png)

### 模块定位与适用角色
- 提示词模板模块用于模板复用、治理与版本维护，面向全部登录用户（权限分层）。

### 典型使用场景
- 创建个人模板提升对话一致性。
- 收藏高频模板并设定全局默认模板（管理员）。
- 通过版本回溯恢复误改模板。

### 功能清单
- 能浏览模板广场，在 `/prompt-templates`，登录用户可用，支持分类/可见性/排序筛选。
- 能新建模板，在 `/prompt-templates/new`，登录用户可用，生成可复用模板。
- 能编辑模板，在 `/prompt-templates/:id`，有权限用户可用，更新模板内容与元数据。
- 能收藏/取消收藏模板，在模板卡片，登录用户可用，便于快速调用。
- 能复制模板，在模板卡片，登录用户可用，快速派生个人版本。
- 能删除模板，在模板卡片，具备权限用户可用，删除后不可恢复。
- 能设为全局默认模板，在模板卡片，管理员可用，新对话默认继承。
- 能查看模板统计与版本、创建版本、回滚版本，在模板详情，具备权限用户可用。

### 操作路径
- 顶部导航进入 `/prompt-templates`。
- 点击“新建模板”或进入模板详情进行编辑。

### 输入/输出与结果表现
- 输入：模板名称、分类、可见性、正文、优先级。
- 输出：可在会话中直接选择的模板资产。

### 权限与可见性
- 普通用户可管理个人模板。
- 管理员可管理公开模板并设置全局默认。

### 约束与边界
- 内置模板不可直接编辑删除（需复制后修改）。

### 常见问题（FAQ）
- Q：模板改动会影响历史会话吗？
- A：不会，历史会话保留当时已应用内容。

## 7. 历史对话 / 会话协作
![历史对话模块截图](/Users/bk27284/Documents/ai-skill-sharing-platform/docs/images/v1.0.0/07-chat.png)

### 模块定位与适用角色
- 对话模块是核心协作面板，承载消息交互、工具调用、文件沙箱、引用治理与导出反馈。

### 典型使用场景
- 多轮协作产出文档、代码或决策结论。
- 在会话中切换模型、模板、激活技能。
- 对会话进行标签管理、导出与批量操作。

### 功能清单
- 能创建、查看、删除会话，在 `/chat/:id?` 与侧栏，登录用户可用，会话生命周期可管理。
- 能发送消息并接收流式回复，在会话主区，登录用户可用，支持中断生成。
- 能对已发送消息进行重新编辑，在会话消息区，登录用户可用，编辑后可基于新内容继续生成。
- 能进行对话路由分叉并切换分支，在会话分支操作区，登录用户可用，支持多路径推演与回看。
- 能上传文件并在会话中引用，在输入区与文件组件，登录用户可用，支持多模态输入。
- 能切换会话模型，在会话头部模型选择器，登录用户可用，后续轮次按新模型执行。
- 能切换会话模板，在模板选择器，登录用户可用，对后续生成上下文生效。
- 能激活/确认/拒绝会话技能，在技能管理面板，登录用户可用，能力边界可控。
- 能查看并管理会话沙箱文件（预览/下载/编辑/重命名/删除/归档下载），在沙箱面板，登录用户可用。
- 能管理会话标签（单条/批量），在侧栏与标签面板，登录用户可用，提升检索效率。
- 能导出会话（单条/批量，Markdown 或 Docx），在导出操作区，登录用户可用，便于归档。
- 能提交反馈并查看历史反馈，在反馈弹窗，登录用户可用，支持质量闭环。
- 能执行上下文压缩，在会话操作区，登录用户可用，降低超长对话负担。
- 能管理引用范围与选中资料，在引用面板，登录用户可用，保障回答引用可控。
- 能在会话中使用文件沙箱，在沙箱面板，登录用户可用，支持文件预览、编辑、下载和归档导出。

### 操作路径
- 顶部导航进入“历史对话” `/chat`。
- 左侧选择会话，右侧进行消息协作与管理。

### 输入/输出与结果表现
- 输入：文本、附件、技能激活、模型/模板切换、标签与引用设置。
- 输出：多轮消息、工具调用结果、沙箱文件、可导出成果。

### 权限与可见性
- 登录用户可访问自身会话范围。
- 管理视图下可只读查看指定会话（管理员入口）。

### 约束与边界
- 长会话在高并发下可能存在渲染延迟，建议使用筛选与压缩。
- 外部工具调用成功率受 MCP 与模型状态影响。

### 常见问题（FAQ）
- Q：会话里切换模型会影响已生成内容吗？
- A：不会，已生成内容保留；切换仅影响后续生成。
- Q：沙箱文件能否批量下载？
- A：支持会话沙箱归档下载。

## 8. 管理后台总览
![管理后台总览截图](/Users/bk27284/Documents/ai-skill-sharing-platform/docs/images/v1.0.0/08-admin-overview.png)

### 模块定位与适用角色
- 管理后台用于系统级配置与治理。

## 12. MCP 管理
![MCP管理模块截图](/Users/bk27284/Documents/ai-skill-sharing-platform/docs/images/v1.0.0/12-admin-mcps.png)

### 模块定位与适用角色
- MCP 管理用于外部工具接入、巡检、工具开关和健康治理，面向超级管理员。

### 典型使用场景
- 新接入一个 MCP 服务并配置超时/重试/熔断参数。
- 定时巡检 MCP 连通性，定位不稳定工具。
- 对单个工具做启停控制。

### 功能清单
- 能创建/编辑/删除 MCP，在 `/admin/mcps`，Admin 可用，维护接入配置。
- 能启停 MCP，在列表操作区，Admin 可用，控制整体可用性。
- 能测试单个 MCP 或批量测试，在测试操作区，Admin 可用，输出成功/失败与耗时。
- 能刷新 MCP 工具列表，在工具抽屉，Admin 可用，同步新增/删除/未变工具。
- 能启停单个工具，在工具抽屉，Admin 可用，细粒度控制调用范围。
- 能配置自动巡检频次，在巡检配置栏，Admin 可用，形成持续健康检查机制。

### 操作路径
- 进入 `/admin/mcps`。
- 在列表与工具抽屉中维护配置与状态。

### 输入/输出与结果表现
- 输入：连接配置 JSON、超时/重试/熔断参数、角色访问级别。
- 输出：MCP 状态、工具清单、巡检结果。

### 权限与可见性
- 仅 Admin 可见。

### 约束与边界
- 依赖外部工具协议与网络可达性。
- 配置错误会导致测试失败或工具不可调用。

### 常见问题（FAQ）
- Q：刷新工具后为什么少了工具？
- A：源端工具定义发生变化，平台按最新结果同步。

- 发布新版本并切换运行实例。

### 功能清单
- 能创建流程定义，在 `/admin/workflows`，Admin 可用，形成可维护流程实体。
- 能按作用域筛选流程定义，在流程列表，Admin 可用，区分项目/需求流程。
- 能废弃或重新启用流程定义，在列表操作，Admin 可用，控制可用性。
- 能创建草稿版本，在流程编辑页，Admin 可用，支持从已选版本分支。
- 能维护节点（增删改、拖拽排序、绑定 Skill、输出类型），在流程编辑页，Admin 可用，形成可执行节点链路。

### 操作路径
- 进入 `/admin/workflows` 创建或选择定义。

### 输入/输出与结果表现
- 输入：流程名称/编码、节点配置、版本操作。
- 输出：可绑定到项目或需求的流程版本。

### 权限与可见性
- 仅 Admin 可见。

### 约束与边界
- 已发布版本不可直接改写，需创建草稿分支再修改。

### 常见问题（FAQ）

## 14. 会话管理 / 审计
![会话管理审计模块截图](/Users/bk27284/Documents/ai-skill-sharing-platform/docs/images/v1.0.0/14-admin-audit.png)

### 模块定位与适用角色
- 会话管理与审计用于运行质量治理、异常监控、轨迹追溯与回放，面向超级管理员。

### 典型使用场景
- 查询异常会话并定位问题轮次。
- 查看会话时间线与 trace 轨迹。
- 做只读回放用于问题复盘。

### 功能清单
- 能按用户、模型、异常状态、MCP 工具筛选会话，在 `/admin/audit/conversations`，Admin 可用，定位目标会话。
- 能查看会话摘要和异常类型，在会话列表，Admin 可用，快速识别风险会话。
- 能查看会话轮次时间线，在会话详情页，Admin 可用，了解每轮状态与耗时。
- 能查看指定轮次 JSON 明细，在轮次详情页，Admin 可用，支持深入排查。
- 能执行 Trace 只读回放，在会话详情页，Admin 可用，验证事件链完整性。
- 能查看异常监控指标（异常率、追溯覆盖率、回放成功率），在“异常监控”标签页，Admin 可用。
- 能按日/周/月切换监控窗口，在异常监控页，Admin 可用，观察趋势变化。

### 操作路径
- 进入 `/admin/audit/conversations`。
- 在“会话管理/异常监控”两个标签页中切换分析。

### 输入/输出与结果表现
- 输入：筛选条件、窗口周期、trace 回放动作。
- 输出：异常分布、轮次轨迹、审计明细与回放结果。

### 权限与可见性
- 仅 Admin 可见。

### 约束与边界
- 回放为只读，不会触发真实执行副作用。

### 常见问题（FAQ）
- Q：异常会话是实时标记的吗？
- A：异常标记受事件采集与规则计算周期影响，通常接近实时。
